# CLI Tool

- Status: Project-owned

## CLI Boundary

Ohrisk exposes a Node.js CLI for license-risk scanning and reporting.

## Commands

- `ohrisk scan`
- `ohrisk ci`
- `ohrisk diff <ref>`
- `ohrisk explain <expr>`
